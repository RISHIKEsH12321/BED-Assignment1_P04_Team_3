Parse incoming request bodies in a middleware before your handlers, available under the req.body property.
bcryptjs
Optimized bcrypt in JavaScript with zero dependencies. Compatible to the C++ bcrypt binding on node.js and also working in the browser.
dotenv
Dotenv is a zero-dependency module that loads environment variables from a .env file into process.env. Storing configuration in the environment separate from code is based on The Twelve-Factor App methodology.
express
Express is a minimal and flexible Node.js web application framework that provides a robust set of features for web and mobile applications.
express-fileupload
Simple express middleware for uploading files.
joi
The most powerful schema description language and data validator for JavaScript. joi lets you describe your data using a simple, intuitive, and readable language.
jsdom
jsdom is a pure-JavaScript implementation of many web standards, notably the WHATWG DOM and HTML Standards, for use with Node.js. In general, the goal of the project is to emulate enough of a subset of a web browser to be useful for testing and scraping real-world web applications.
jsonwebtoken
An implementation of JSON Web Tokens.
This was developed against draft-ietf-oauth-json-web-token-08. It makes use of node-jws
mssql
Microsoft SQL Server client for Node.js
nodemon
nodemon is a tool that helps develop Node.js based applications by automatically restarting the node application when file changes in the directory are detected.
jest
By ensuring your tests have unique global state, Jest can reliably run tests in parallel. To make things quick, Jest runs previously failed tests first and re-organizes runs based on how long test files take.
axios
Alloy is the integration development platform that makes it simple and fast for SaaS companies to launch critical user-facing integrations.
google-ai/generativelanguage
Generative Language API client for Node.js
googleapis
Node.js client library for using Google APIs. Support for authorization and authentication with OAuth 2.0, API Keys and JWT tokens is included.
